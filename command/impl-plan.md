Use the jira-enhance skill in **impl-plan** mode.

Analyze the acceptance criteria for the specified Jira ticket (or auto-detect from branch name), explore the relevant codebases to understand existing patterns, and generate a detailed implementation plan.

Post the implementation plan as a comment on the Jira ticket, supplementing any existing content under "Implementation Details & Additional Resources."
